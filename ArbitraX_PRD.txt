1. Visão Geral

Produto
ArbitraX — Plataforma de Arbitragem Virtual via WhatsApp (1-on-1) com IA

Slogan
Justiça privada, acessível e rápida — direto no seu WhatsApp.

Problema
Judiciário brasileiro: 80M processos pendentes, 4–7 anos para sentença. Arbitragem tradicional é boa mas cara (R$15–40K) e burocrática. PMEs desistem de cobrar dívidas. Não existe solução que combine velocidade da IA com validade jurídica da arbitragem num canal universal.

Solução
ArbitraX: arbitragem 100% virtual. Partes conversam 1-on-1 com bot no WhatsApp (Meta Cloud API, sem Twilio, sem grupo). Árbitro opera pelo painel web. IA auxilia análise e gera projetos de sentença, sempre validados por árbitro humano. Lei 9.307/96.

Diferenciais Arquiteturais
WhatsApp 1-on-1 (sem grupo): sigilo reforçado, UX simples, sem confusão
Meta Cloud API (sem Twilio): 1.000 conversas/mês gratis, economia de R$2–5K/mês
Árbitro 100% painel web: interface profissional, sem limitações do WhatsApp
IA + Humano: velocidade da IA com autoridade do árbitro
Custo 5–10x menor que arbitragem tradicional
Resolução em 15–45 dias (vs 4–7 anos)

LGPD: consentimento, portabilidade, retenção 5 anos

6. Jornadas do Usuário

6.1 Requerente
Acessa site → preenche formulário → paga R$350 via PIX
Bot envia WhatsApp (1-on-1): “Caso criado. Convite enviado ao requerido.”
Quando requerido aceita e ambos assinam: bot avisa
Bot: “10 dias para enviar petição. Envie tudo aqui neste chat.”
Requerente envia textos e docs no chat individual
Se IA pede mais provas: bot solicita especificamente
Sentença publicada: bot envia PDF + código individualmente
Total: 22 dias, custo R$2.300. Sentença válida.

6.2 Requerido
Recebe WhatsApp do bot (1-on-1): convite + link
Clica → página web → aceita → assina compromisso
Bot: “Aguardando petição do requerente.”
Bot: “10 dias para contestar. Envie aqui.”
Envia contestação e provas no chat individual
Recebe sentença individualmente via WhatsApp

6.3 Árbitro (100% Painel Web)
Recebe notificação no painel: caso designado
Acessa caso: lê peças, provas, timeline
IA gera projeto → badge “Sentença pendente” no dashboard
Abre projeto. Lê. Sugere ajuste na fundamentação.
IA gera V2. Aprova. Ratifica.
Honorários creditados.
